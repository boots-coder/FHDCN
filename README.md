# FHDCN

Source code for the paper: **"FHDCN: Feature Hierarchy-Based Dynamic Cascade Network with a novel adaptive ensemble classifier for High-Dimensional Small-Sample cancer classification"**

## Requirements

- Python 3.8+
- PyTorch, scikit-learn, NumPy, SciPy

```bash
pip install torch numpy scikit-learn scipy
```

## Quick Start

```bash
python example.py
```

This runs the full FHDCN pipeline on the ALLAML dataset.
